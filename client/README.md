# 🚀 James Balito - Full Stack Developer Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full-Stack Developer. Built with cutting-edge web technologies and featuring smooth animations, dark mode, and an infinite tech stack carousel.

![Portfolio Demo](https://via.placeholder.com/800x400?text=Portfolio+Demo)

## ✨ Features

- 🎨 **Modern UI/UX Design** - Clean and professional interface with smooth animations
- 🌓 **Dark/Light Mode** - Seamless theme switching with persistent user preference
- 🎭 **GSAP Animations** - Stunning entrance animations and page transitions
- 🔄 **Infinite Carousel** - Smooth, endless scrolling tech stack showcase
- 📱 **Fully Responsive** - Perfect display on all devices (mobile, tablet, desktop)
- ⚡ **Performance Optimized** - Fast loading and smooth interactions
- 🎯 **Typewriter Effect** - Dynamic text animation for hero section
- 🎨 **CSS Variables** - Easy theme customization and maintenance

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **GSAP** - Professional animations
- **Lucide React** - Beautiful icons

### Development Tools
- **Vite** - Build tool
- **ESLint** - Code linting
- **Prettier** - Code formatting
